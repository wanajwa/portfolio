EPine – Smart Pineapple Farming System

Kotlin | SULAM Project

EPine is a mobile application developed to help pineapple farmers monitor and manage their farms efficiently. Farmers can report diseases, track sales records, view profit summaries, and access promotional information. The system improves farm productivity and supports better decision-making in pineapple cultivation.

Features

  • Login system for farmers and vendors
  • Report pineapple diseases with photos
  • Track and add daily sales records
  • View monthly and yearly profit summaries
  • View product and promotion details
  • Navigation between app fragments/pages
  • Secure logout functionality

Dashboard Navigation Design

Collaboration with SULAM

EPine was developed in collaboration with Service Learning Malaysia (SULAM), a university initiative aimed at using technology to benefit society. This partnership ensured that the system addresses real community needs for pineapple farmers.
